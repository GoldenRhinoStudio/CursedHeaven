## Controls

### Keyboard and mouse

* W - Up
* A - Left 
* S - Down
* D - Right
* Q - Ability 1
* E - Ability 2
* P - Basic attack

### Xbox gamepad

* Left joystick - Move
* A - Choose / Accept
* Y - Ability 1
* B - Ability 2
* X - Basic attack
